# SPA Atividade 3

Este projeto demonstra uma Single Page Application (SPA) utilizando JavaScript modular, templates e manipulação de DOM.

## Funcionalidades

- SPA básica com Home, Sobre e Contato
- Formulário de contato funcional
- Navegação sem recarregar a página
- Estrutura modular de JS

## Estrutura do Projeto

- `index.html` — Página principal
- `css/style.css` — Estilos
- `js/main.js` — Arquivo principal
- `js/dom.js` — Funções de manipulação do DOM
- `js/templates.js` — Templates HTML
- `images/` — Imagens utilizadas
